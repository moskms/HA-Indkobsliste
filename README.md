# HA Indkøbsliste

Home Assistant add-on repository til "Indkøbsliste" – en indkøbsliste-app med stemmeinput og automatisk butiksopslag baseret på GPS.

## Installation

1. Gå til Home Assistant: **Settings → Apps → App Store**
2. Klik på de tre prikker (⋮) øverst til højre → **Repositories**
3. Tilføj denne URL: `https://github.com/DIT-GITHUB-BRUGERNAVN/HA-Indkobsliste`
4. Find "Indkøbsliste" under de tilgængelige apps, og installer den

## Indhold

- `indkobsliste/` – selve add-on'et (FastAPI-backend + web-frontend)
