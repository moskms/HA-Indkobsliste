# Indkøbsliste

En indkøbsliste med stemmeinput, der automatisk kan foreslå butikker nær din nuværende position.

## Funktioner

- Tilføj varer med stemmeinput (dansk)
- Afkryds/slet varer
- Opret faste butikker manuelt, eller lad appen foreslå butikker automatisk via GPS
- Webhook-endpoint til brug i Home Assistant-automationer (fx notifikation/TTS når du krydser ind i en butiks geofence-zone)

## Brug

Efter start, åbn web-grænsefladen via "OPEN WEB UI"-knappen, eller direkte på:

```
http://<din-ha-adresse>:8000/app/index.html
```
